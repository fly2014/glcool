mysql压缩版安装

1、解压到一个目录,修改配置文件

my-default.ini
basedir =E:\ProgramFiles\mysql-5.6.43-winx64
datadir =E:\ProgramFiles\mysql-5.6.43-winx64\data
port =3306

管理员打开CMD命令

mysqld -install
mysqladmin -u root -p password malong#修改密码
mysqld -remove
net stop mysql
mysql -u root -p
